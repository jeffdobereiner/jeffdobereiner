# Jeffrey (Jeff) Dobereiner

**Space Engineer | NASA Contractor | PhD in Anthropology | MS in Space Systems (in progress)**

Welcome — I’m Jeff Dobereiner, a space-nerd and engineer supporting NASA’s Space Communications and Navigation (SCaN) program. My work spans network architectures, space mission support planning, and defining the lunar communications architecture that will support NASA’s planned Moon base.
